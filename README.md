# VNV-TVApp
Fully-operatable WinForm application for TV providers

Completed as course progect at Vinnytsia IT Academy.

Used technologies/techniques:
- DOT.NET Framework
- Entity Framework (Code First approach) 
- Windows Forms

- exception handling
